# Full-Stack-Blog 

## 技术栈
- Koa@2 + MongoDb + Passport 组织后端逻辑
- React + Dva + Antd 结合实现单页应用
- Draftjs 可视化编辑
- webpack-dev-server 搭建开发环境
- pm2 自动部署

![Editor](./doc/img/editor_screenshot.png)

## 环境准备
```
node.js 7.0+
mognodb 3.0+
```

## 启动方法
```
npm install
npm run server
npm start
```
